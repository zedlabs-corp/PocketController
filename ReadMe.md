# 🎮 PocketController PC Client

**PocketController** is a premium, high-performance bridge that transforms your mobile device into a versatile virtual game controller for your Windows PC. Experience seamless gaming with professional-grade input emulation and ultra-low latency.

---

## ✨ Key Features

- 🕹️ **Pro-Grade Emulation**: Implements native Xbox 360 and DualShock 4 controller profiles via the ViGEm framework for 100% game compatibility.
- ⌨️ **Input Simulation**: Dynamic switching between physical gamepad emulation and Keyboard/Mouse simulation.
- ⚡ **Ultra-Low Latency**: High-performance WebSocket architecture optimized for competitive gaming with real-time jitter compensation.
- 👥 **Local Multiplayer**: Supports multiple simultaneous mobile connections for split-screen and couch co-op gaming.
- 📉 **Performance Monitoring**: Real-time tracking of network latency and connection stability directly in the client.
- 📥 **System Tray Integration**: Runs quietly in the background, minimizing to the system tray to stay out of your way while you play.

---

## 🚀 Installation

Getting started is easy! You don't need to manually configure complicated server environments.

1.  **Download**: Head over to our [**Releases Page**](https://github.com/zedlabs-corp/PocketController/releases) and download the latest `PocketController-Setup.exe`.
2.  **Install**: Run the installer and follow the on-screen instructions.
3.  **Drivers**: The setup will automatically prompt you to install the necessary **ViGEmBus** drivers if they are not already present on your system.

---

## 🛡️ Note on Windows SmartScreen

As PocketController is an independent project, Windows may show a **"Windows protected your PC"** warning during installation. 

**This is normal!** Because the installer is not yet digitally signed by a major publisher, Microsoft SmartScreen flags it. To proceed:
1. Click **"More info"** on the blue warning screen.
2. Click **"Run anyway"** to start the installation.

---

## 🛠️ How to Use

1.  **Launch**: Open the PocketController PC Client from your Desktop or Start Menu.
2.  **Connect**: Note the **IP Address** displayed on the main dashboard.
3.  **Pair**: Open the **PocketController Mobile App** on your smartphone and enter the IP address (or scan the QR code if available).
4.  **Play**: Select your preferred layout and start gaming!

> [!TIP]
> Ensure both your PC and Mobile device are connected to the same Wi-Fi network for the best performance and easiest discovery.

---

## 📋 Requirements

- **OS**: Windows 10/11 (64-bit)
- **Network**: Stable Wi-Fi/Local Network connection.
- **Drivers**: ViGEmBus (included in installer).
